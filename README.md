# QRESFEM
Replication archive for "Quantal Response Equilibrium Strategy Frequency Estimation"

Here you will find replication code for the above-mentioned paper.

Copy everything into a folder called "code_SFEMQRE" and then put this folder into another folder. Se this second folder to your working directory in R and then run the .R scripts in numerical order.

These R scripts require the following libraries

* tidyverse
* rstan
* bridgesampling
* loo
* kableExtra
* latex2exp
* Cairo
* viridis
